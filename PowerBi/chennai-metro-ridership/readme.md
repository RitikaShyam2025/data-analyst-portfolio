# 🚇 Chennai Metro Ridership & Revenue Dashboard

An interactive Power BI dashboard built to analyze Chennai Metro ridership, ticket revenue, station performance, and payment trends using a Star Schema data model.

---

## 📌 Project Overview

This dashboard provides business insights into Chennai Metro operations by analyzing:

- Total Revenue
- Total Trips
- Total Passengers
- Average Fare
- Revenue per Passenger
- Revenue by Station
- Revenue by Ticket Type
- Revenue by Payment Mode
- Monthly Revenue Trend
- Station Performance Ranking

---

## 🛠️ Tools & Technologies

- Power BI Desktop
- Power Query
- DAX
- Star Schema Data Modeling
- Microsoft Excel

---

## 📊 Data Model

The project follows a Star Schema.

### Fact Table
- Metro Trips

### Dimension Tables
- Date
- Station
- Ticket
- Payment
