📊 Interactive E-Commerce Sales Dashboard | Power BI
📌 Project Overview

This project is an Interactive E-Commerce Sales Dashboard built using Microsoft Power BI. The dashboard provides business insights by analyzing sales performance, customer behavior, regional sales, payment methods, and product performance. It is designed to help users make data-driven decisions through interactive visualizations.

🎯 Objectives

Analyze overall sales performance.
Monitor profit and profit margin.
Track customer orders.
Compare sales across different regions.
Analyze payment methods and product categories.
Enable users to explore detailed insights using interactive features.
